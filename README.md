# vue_shop

> 基于Vue的电商后台管理项目，前后端分离，包括权限管理，用户管理，商品管理，订单管理等功能。

## 前端技术栈

- Vue
- Vue_Router
- Element UI
- Axios
- Webpack
- Echarts


## 如何启动

``` bash
# 安装依赖
npm install

# 启动项目
npm run dev
# 项目打包
npm run build
npm run build --report
```
